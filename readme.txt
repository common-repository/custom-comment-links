Customize comment links on your site. Control comment author's URL, remove links from comments. Disable these options for privileged users.

== Description ==

Ever wish you could remove links inside comments, or their author's website?

With this simple plugin you can customize how your site's comment links are shown for author and content.
Enable this plugin and select if:

* Users with post editing capabilities are affected by the settings.
* The comment author's website link is loaded.
* Any links in the comments are unset or moved next to the link text inside parenthesis.
